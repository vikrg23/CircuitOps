arc_delay_prediction_utils.merge_tran_cell(pin_pin_df, pin_df, cell_df)¶

Return a pin to pin df after merging tran and cell type columns cell_types are coded as int to make it ML friendly.

Parameters:

  • pin_pin_df (Pandas datframe of type pin_pin_df) – Required. Original pin to pin df to be processed

  • pin_df (Pandas datframe of type pin_df) – Required. Original pin properties df to be processed

  • cell_df (Pandas datframe of type cell_df) – Required. Original cell properties df to be processed

Returns:

pin to pin df with pin_tran, cell_type, cell_type_coded columns

Pandas df
